# Tasca 01

## 📖 Descripció
En aquesta tasca es treballa la cerca d’informació sobre els superordinadors més potents del món.

## 🎯 Objectiu
Investigar quins són actualment:
- El primer superordinador mundial
- El segon superordinador mundial

## 🖥️ Resultats

### 1️⃣ Frontier
Actualment, el superordinador més potent del món és **Frontier**.

- Rendiment: 1.102 exaFLOPS
- Ubicació: Estats Units
- Funció principal: càlcul científic avançat i intel·ligència artificial.

### 2️⃣ Fugaku
El segon superordinador més potent és **Fugaku**.

- Rendiment: 442 petaFLOPS
- Ubicació: Japó
- Desenvolupat per: RIKEN i Fujitsu
